# Patrimonio · Bolivia 📊
App personal de finanzas con tipo de cambio BCB en tiempo real.

## Cómo publicar en GitHub Pages (5 minutos)

### Paso 1 — Crear cuenta en GitHub
Si no tenés cuenta: https://github.com/signup (es gratis)

### Paso 2 — Crear repositorio
1. Ir a https://github.com/new
2. En "Repository name" escribir: `patrimonio-bo`
3. Seleccionar **Public**
4. Click en **Create repository**

### Paso 3 — Subir los archivos
En la página del repositorio vacío:
1. Click en **uploading an existing file**
2. Arrastrar estos 4 archivos:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Click en **Commit changes**

### Paso 4 — Activar GitHub Pages
1. Ir a **Settings** (pestaña del repositorio)
2. Scroll hasta **Pages** (menú izquierdo)
3. En "Source" seleccionar **Deploy from a branch**
4. En "Branch" seleccionar **main** → **/ (root)**
5. Click **Save**

### Paso 5 — Obtener la URL
Esperar ~1 minuto, luego la URL aparece en Settings → Pages:
```
https://TU_USUARIO.github.io/patrimonio-bo/
```

### Paso 6 — Instalar en el celular
1. Abrir la URL en **Chrome** (Android) o **Safari** (iPhone)
2. Android: menú ⋮ → "Añadir a pantalla de inicio" → "Instalar"
   iPhone: botón compartir □↑ → "Añadir a pantalla de inicio"
3. Aparece el ícono dorado en tu pantalla

## Características
- 💰 Ingresos, gastos, activos y deudas
- 🏦 Tipo de cambio BCB en tiempo real
- 💱 Toggle Bs ↔ USD
- 📊 Análisis y gráficos
- 📥 Exportar a Excel (5 hojas)
- 💾 Datos guardados localmente en el teléfono
- 📴 Funciona sin internet (offline)
- 📱 Instalable como app nativa (PWA)
